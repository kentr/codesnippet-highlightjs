# Highlight.js

This repo simply contains the contents of a [highlight.js](https://highlightjs.org) package with a custom language set.

Highlight.js version: `9.13.1`.

## Languages included

* apache
* applescript
* bash
* css
* diff
* handlebars
* http
* ini
* javascript
* json
* markdown
* nginx
* perl
* php
* plaintext
* python
* ruby
* scss
* shell
* sql
* twig
* xml
* yaml
